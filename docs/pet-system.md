# 펫 시스템

## 개요

마을에서 각자의 역할을 수행하는 존재들.
낮에 일하고 밤에 잠에든다. 다양한 유형 (농사, 채굴, 버프, 건설 등).
마을에 배치 가능한 펫 수 제한 → 마을 확장으로 최대 수 증가.

---

## 행동 패턴

### 밤
- 모든 펫 수면 상태 (특수 특성/버프 제외)
- 눈 감은 모델 + 잔잔한 숨쉬기 애니메이션
- 콧소리 사운드 + Zzz 이펙트
- 불면 상태 펫은 낮과 동일 행동

### 낮
1. **일 시작**: 빈 일터 중 자기가 할 수 있는 곳 탐색 → 이동 → 작업
   - 두 가지 이상 가능하면 랜덤 선택
2. **피로**: 2분~4분 후 땀 VFX → 휴식 시작
3. **휴식**: 쉼터 중 빈 자리로 이동
   - 쉼터 없으면 → "산책" 상태 (마을 이곳저곳 배회)
   - 쉼터 있어도 25% 확률로 산책 선택
   - 산책 중 다른 펫 만나면 멈춰서 대화 (말풍선 VFX)
4. **휴식 종료**: 1분~1분30초 후 다시 일 탐색
   - 이전 일터 50% 선호 (자리 비어있으면)

> 활동 루틴 시간과 확률은 추후 변경 가능

---

## 무게

- 부화 시 랜덤 결정: 표준무게 × 0.7 ~ 2.0배
- 무게 배수만큼 모델 크기 스케일
- 작업 효율 = (1 + 무게배수) / 2
  - 예: 12kg (표준 10kg) → 1.2배 → 효율 1.1배

---

## 레벨 & 경험치

- 최대 레벨: 10
- 레벨 10 시 레벨 1 대비 약 2배 효율
- 경험치: 매 초 자동 획득
- 특정 펫/건물이 경험치 가속 가능
- 레벨업 시 경험치 0으로 초기화, 다음 레벨 요구량 증가

## 유형

| 유형 | 설명 |
|------|------|
| 농사 | 농원에서 농사를 지으며 일하는 펫 |
| 채굴 | 채굴장에서 채굴하여 일하는 펫 |
| 버프 | 다른 펫 또는 플레이어에게 버프를 거는 펫 |
| 건설 | 건물 업그레이드 작업을 담당하는 펫 |
| 기타 | 기타 |

---

## 사이즈

내부 시스템 변수. 모자 크기 통일, 크기 분포 밸런스용.
Small을 표준 기준으로 한다.

| 사이즈 | 설명 | 표준 무게 |
|--------|------|----------|
| Small | 플레이어 하체 정도 | 5kg |
| Medium | 플레이어 목까지 / 절반 부피 | 10kg |
| Big | 플레이어와 비슷 | 25kg |
| Huge | 플레이어의 약 1.5배 | 60kg |
| Ultra | 플레이어의 약 2배 | 150kg |

---

## 종족

펫 종류에 귀속되는 특성. 그 자체로 특별한 능력 없음 (추후 추가 가능).
특수 펫/건물에게 영향 받는 특성.
예: 모든 해골 종족 펫에게 이동속도 버프를 주는 무덤 건물.
