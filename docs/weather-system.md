# 날씨 시스템

## 개요

랜덤한 시간에 랜덤한 날씨가 적용된다.
날씨에 따라서 특수한 효과가 발생하거나 적용된다.

- "글리치", "일식"같은 낮은 확률로 등장하는 특수한 날씨들도 존재
- 날씨는 모든 서버 공통
- 동시에 두개 이상 적용 불가
- 지속시간: 7분~10분 (추후 변경 가능)
- 쿨다운: 30분~50분 (추후 변경 가능)
- 날씨 시작/종료 시 화면 상단 메시지 출력

## 날씨 테이블

| 날씨 | 확률 | 설명 | 효과 | 알림 메시지 |
|------|------|------|------|------------|
| Sunny (맑음) | 상시 | 해가 뜨고 하늘이 맑음. 기본 날씨 | 없음 | 없음 |
| Rain (비) | 높음 | 비가 내리며, 빗소리가 들림 | 농사 속도 50% 증가 | 비는 대지를 적시고 생명을 깨웁니다. 농사 효율이 50% 증가합니다! |
| Frost (눈) | 높음 | 눈이 내림. 캐릭터가 오들오들 떨림 (Motor6D) | 펫 버프 지속시간 소모 안 됨 | 차가운 기운이 시간을 얼려버립니다. 펫의 버프 지속시간이 소모되지 않습니다! |
| Thunder Storm (뇌우) | 낮음 | 비 + 랜덤 위치 번개 | 번개 맞은 펫 → "충전된" 돌연변이 획득 | 폭풍우 속 번개의 힘이 깃듭니다! 번개에 맞은 펫이 영구적으로 더 강력해집니다! |

> 추후 추가 예정: Glitch (글리치), Eclipse (일식) 등 특수 날씨
